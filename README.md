# 🐕 StrayGuard — India Stray Dog Management System

> A data-driven platform for analyzing stray dog populations, vaccination coverage, rabies incidents, and Animal Birth Control (ABC) programs across India.

## 📖 Overview

StrayGuard is a Final Year Project designed to provide a comprehensive view of India's stray dog ecosystem. The platform aggregates and visualizes state-wise data, helping identify high-risk regions, monitor vaccination coverage, analyze rabies trends, and evaluate ABC program effectiveness.

The goal is to support better decision-making through data analytics and interactive visualizations.

---

## ✨ Key Features

### 📊 State-Wise Analytics
- Detailed data for all Indian states
- Stray dog population statistics
- Rabies and bite incident tracking
- Municipal budget analysis

### 💉 Vaccination Monitoring
- Vaccination coverage by state
- Comparative analysis across regions
- Identification of low-coverage areas

### 🐾 ABC Program Tracking
- Animal Birth Control (ABC) program monitoring
- Sterilization statistics
- Program effectiveness insights

### 📈 Analytics Dashboard
- Interactive charts and graphs
- Trend analysis and comparisons
- High-risk state identification

### 🔍 Search & Filter System
- Search states instantly
- Filter by risk category
- Sort data dynamically

### 📱 Responsive Design
- Mobile-friendly interface
- Optimized for desktop and tablet devices
